📝 Simple Task Manager (Sistema de Tarefas)
Uma aplicação simples em Python para gerenciamento de tarefas diárias.

🚀 Funcionalidades
01. Cadastrar Tarefa: Adicione uma nova descrição para sua tarefa.
02. Listar Tarefas: Visualize todas as tarefas armazenadas no sistema.

🛠️ Tecnologias Utilizadas
Linguagem: Python

Bibliotecas:
01. Flask: micro-framework para criação da pagina web.
02. OS: interagir com o sistema operacional (ex: os.system('cls')).
